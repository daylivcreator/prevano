4. Données personnelles et RGPD

4.1 Responsable du traitement

Prevano SAS est responsable du traitement de vos données personnelles collectées via le site prevano.fr.

4.2 Données collectées

Nous collectons les données suivantes :

  • Données de compte : prénom, adresse email, mot de passe (haché avec bcrypt) ;
  • Données de simulation : âge, salaire, épargne mensuelle, âge de départ souhaité, statut professionnel — traitées en mémoire et non stockées de façon permanente pour les utilisateurs gratuits ;
  • Données de paiement : traitées exclusivement par Stripe — Prevano ne stocke aucune donnée bancaire ;
  • Données de navigation : adresse IP, logs d'accès (conservés 30 jours à des fins de sécurité).

4.3 Base légale et finalités

  • Exécution du contrat : gestion du compte, fourniture du service, facturation ;
  • Intérêt légitime : sécurité du service, prévention des fraudes ;
  • Consentement : communications marketing (opt-in explicite requis).

4.4 Durée de conservation

  • Données de compte : durée de la relation contractuelle + 3 ans après la suppression du compte ;
  • Données de facturation : 10 ans (obligation légale comptable) ;
  • Logs de sécurité : 30 jours.
